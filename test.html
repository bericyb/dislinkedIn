üîß Technical Details

How It Works:
  • Content script injects dislike buttons into LinkedIn's action bars
  • Uses LinkedIn's post URN (data-urn attribute) for unique identification
  • Stores dislike data locally using Chrome storage API
  • Mock backend simulates server-side functionality
Post Identification Example:
data-urn="urn:li:activity:7380291595671199744"

This unique identifier allows the extension to track dislikes per post

üõ† Troubleshooting

If buttons don't appear:
  • Check that the extension is enabled in chrome://extensions/
  • Refresh the LinkedIn page
  • Open browser console (F12) and look for DislinkedIn messages
  • Make sure you're on linkedin.com (not the mobile version)
Icon Issues:

If you see icon errors, you can temporarily remove the "icons" section from manifest.json for testing
